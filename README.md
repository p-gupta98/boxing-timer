# Boxing Timer

A boxing round timer built with React. Designed for fighters and coaches to manage training sessions with precision.

## Features
- Customizable rounds, round length, and rest time
- Fight and rest phase transitions with automatic round tracking
- Visual countdown ring that shrinks in real time
- Bell sounds for round start, warning, and end
- AI Cornerman — ask a boxing coach anything between rounds

## Tech Stack
- React
- Vite
